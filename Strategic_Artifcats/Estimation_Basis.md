# Estimation Basis

This document records the assumptions behind the original estimates for project tasks. Estimates depend on these assumptions being valid.

| Task ID | Original Estimate | Key Assumption |

| BAN-001 | 5 days | The nurse user interface will use standard login and does not require biometric authentication. |
| BER-001 | 7 days | The AI training data collected from Cairo hospitals is clean and does not require major preprocessing. |
| SYS-002 | 4 days | The system will use an existing database structure and will not require designing a new database from scratch. |

## Assumptions Summary
1. The nurse UI will not require biometric authentication.
2. The AI training data from Cairo is assumed to be clean.
3. The system will use an existing database structure.
