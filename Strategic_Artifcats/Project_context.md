The main goal is to improve healthcare services by using predictive modeling, secure data integration, mobile tools for nurses, and organized project management. Each location has a clear responsibility, which helps divide the work in a structured way.

This project also helps avoid common software problems that usually happen because of poor planning, weak communication, or missing documentation. In GCHA, we use Jira to organize the work into Epics and user stories so we can clearly track progress. We also use GitHub to store documents and monitor any updates or changes.

By focusing on organization and proper documentation, the project reduces confusion and ensures that everyone follows the required standards. This makes the system more reliable and easier to maintain in the future.
