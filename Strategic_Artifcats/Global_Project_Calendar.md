## Schedule Assumptions

1. **Dead Zone Lag:** A 1-day Lag has been added to every inter-site handover
   to account for the Dead Zone communication delay between global sites.

2. **Cairo Weekend Conflict:** All Cairo-Berlin collaborative tasks must be
   scheduled Monday–Thursday only (the only days both sites are working).

3. **Golden Window:** London–Bangalore synchronous window is 2 hours/day
   (10:00–12:00 GMT). All critical approvals must occur during this window.

4. **Holiday Buffer:** A 2-day contingency reserve has been added around each
   major holiday listed above.
